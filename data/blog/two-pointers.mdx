## Ex 1:

Given an array of sorted integers, return indices of the two numbers such that they add up to a specific target. There should be one solution

```javascript
const twoSum = function (numbers, target) {
  let start = 0,
    end = numbers.length - 1,
    total

  while (start < end) {
    total = numbers[start] + numbers[end]
    if (total < target) {
      start++
    }
    if (total > target) {
      end--
    }
    if (total === target) {
      return [start + 1, end + 1]
    }
  }
  // In case there is no solution
  return []
}
```

Complexity: O(n)

## Ex 2:

(Valid Palindrome)[https://leetcode.com/problems/valid-palindrome/]

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers.

Given a string s, return true if it is a palindrome, or false otherwise.

Version 1: (Easier, use more mem)

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        fmtString = ""

        for c in s:
            if c.isalnum():
                fmtString += c.lower()
        return fmtString == fmtString[::-1]
```

Complexity: O(n)

Version 2: (Use less memory)

```python
class Solution:
  def isPalindrome(self, s: str) -> bool:
      l, r = 0, len(s) - 1

      while l < r:
          while l < r and not s[l].isalnum():
              l += 1
          while r > l and not s[r].isalnum():
              r -= 1

          if s[l].lower() != s[r].lower():
              return False
          l, r = l + 1, r - 1

      return True
```

Complexity: O(n)
